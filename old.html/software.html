Public software and resources


  • iSeeRNA: accurate and extra-fast lincRNA/mRNA classifier


  • NAMS webserver: coding assessment with functional annotations for plant RNA


  • Sebnif: comprehensive novel lincRNA filter pipeline for RNA-seq data


  • Methy-Pipe: integrated pipeline for BS-seq data analysis


  • BSviewer: lightweight nucleotide-level visualizer for bisulfite sequencing data


  • GeneCT: Cancerous status and tissue origin classifier for pan-cancer biopies


  • LncFunTK: functional lncRNA identification and annotation pipeline


  • FetalQuant: noninvasive fractional fetal DNA concentration deduction


  • FEMER: noninvasive reconstruction of placental methylome


  • ODS: the Online Diagnosis System for Sanger sequencing based genetic testing


  • YY1TargetDB: centralized database for YY1 target information


  • mTFkb: fundamental annotation of mouse transcription factors


  • SKmDB: an integrated database of next generation sequencing information in skeletal muscle.
